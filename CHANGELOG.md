## [1.1.0] - 2025-01-05

### Features
- Major refactoring for performance, security, and code quality improvements
- Enhanced toggle button with dynamic icon and interactions
- Added professional landing page

### Bug Fixes
- Fix critical UX issues with toggle interactions
- Fix grid visibility toggle hiding controls
- Fix resize handle pointer events

- Clean up repository structure

- Improve overall code organization

## [1.0.0] - Initial Release

### Features
- Grid overlay system with responsive breakpoints
- Customizable columns, gutters, and margins
- Color picker with opacity control
- Preset management (save/load)
- Import/Export settings as JSON
- UI scaling support (50-200%)
- Viewport indicator display
- Canvas-based rendering for performance
